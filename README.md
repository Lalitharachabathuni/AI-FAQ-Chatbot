AI FAQ Chatbot

Description

This is a simple AI-powered FAQ chatbot developed as part of the VedGrow Generative AI Internship.

Features

- Answers questions about admissions, fees, courses, and placements
- Maintains memory of the last 5 messages
- Provides fallback responses for out-of-scope questions
- Built using Python

Tools Used

- Python
- Google Colab
